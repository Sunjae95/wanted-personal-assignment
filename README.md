- [x] 돋보기 버튼을 클릭하면 tool tip 출력
- [x] 닫기 버튼을 클릭하면 tool tip 제거
- [x] tool tip 하나만 출력, 다른 가구 선택 시 다른 tool tip 출력
- [x] 하단 상품목록 클릭 시 해당 tool tip 출력
- [x] 입점되어있는 가구, 입점되지 않는 가구 출력
